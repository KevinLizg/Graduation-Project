Introduction

Introduction of how to use this website to improve your math skills

Different Kinds of Practice for each skill and topic

There are three different types of practice.
Some of them could give rewards after you answer questions correctly

Practice

Practices are not graded and are not timed. It includes 5 questions for each practice and it is to help you practice and help you enhance the knowledge of different skills. You could add the questions you answered wrong in Wrong Questions Collection to review and re-do these questions. You could also check the solution by paying coins you earn in quizzes and unite tests.

Quiz

Quizzes are for different skills, it is limited to 2 minutes.It has 10 questions for each quiz. For each question you answer correctly, you could earn 1 coin. After you finish the quiz or time is up, the result of the quiz will show to you, the result will be also recorded into you profile page in order to help you analyze your recent grade changes and change your learning strategy based on the analysis.

Unit Test

There are five unit tests for you to do. The unit tests contain Basic Math, Algebra, Geometry, Statistic and Misc. You can do unit tests to earn coins as well. Unit test is limited to 3 minutes, it has 20 questions for each unit test. For each question you answer correctly, you could earn 2 coins. Unit tests can help you measure how well your entire module is learning. You could also review your recent grade changes as quizzes.

Wrong Question Collection

When you answer the questions in PRACTICE.You could add the questions in this collection so that you can review the questions you have answered. If you have cost the coins to check the solution, you could directly review the solution. If you haven't checked the solution when you add the question, you need to pay coins to check the solution for the question. The wrong question collection can help you learn something new from the past.
